UTC / Z / GMT = Coordinated Universal Time / Zulu Time / Greenwich Mean Time -- Time is based on 0Â° longitude running through the Royal Observatory in Greenwich, London, UK. It is the same all year round and is not affected by Daylight Saving Time (aka Summer Time) clock changes. Using UTC allows weather services and others to avoid time zone conversions.
